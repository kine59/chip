# Chip
Chip' is a recreation of the well-known cookie clicker game. To play it, you simply just click on the cookie.

Please note that I am not associated with Julien Thiennot or DashNet.

If you have any feedback, experience any errors, or have suggestions, please make an issue on the issues board. If you want to improve this project, please do a PR.
